Spanisch-Vokabeltrainer

Statt teure Abonnements bei Sprachlernapps abzuschließen, dachte ich mir, einen eigenen Vokabeltrainer zu machen, um mein Interesse an der spanischen Sprache mit dem Programmieren zusammenzubringen. So das Projekt des Spanisch-Vokabeltrainers entstanden.
Dieses Projekt gibt es in 3 verschiedenen Ausführungen. Meine erste Version ist mit 100% PHP und SQL programmiert. Diese bietet auch ein Backend, welches durch ein Login gesichert ist. Das selbst gestaltete Backend ermöglicht, das Hinzufügen, Löschen und Bearbeiten von Vokabeln (siehe Live).
Die zweite Version ist eine reine Symfony Umsetzung des Projektes. Dort habe ich zusätzlich eine API gemacht, welche beim nächsten Projekt, der App gestaltung, nochmal angesprochen wird. Dort gibt es ein sicheres Login mit Backend und eine Möglichkeit einen Account zu erstellen.(momentan nicht verfügbar)

Frequenzmesser

Noch im ersten Halbjahr meiner Ausbildung sollte ich die Anforderung erfüllen, Daten, die von einem Server kommen, visuell mit Hilfe von Graphen darzustellen.
